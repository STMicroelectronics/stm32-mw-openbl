Purpose

The Open Bootloader package provides an example of an IAP Bootloader that supports OTP and external memories programming.

The Open Bootloader is compatible with STM32CubeProgrammer tool.

In the current release, the Open Bootloader supports USART and USB protocols.

Here is the list of references to the user documents:

  • OTP : OTP management

Main changes

First official version of Open Bootloader that supports OTP partition read/programming and external memories programming.
