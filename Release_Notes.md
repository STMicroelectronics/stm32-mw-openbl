# Purpose

The **STM32N6xx Open Bootloader** package provides an example of an IAP Bootloader that supports OTP and external memories programming.

The Open Bootloader is compatible with STM32CubeProgrammer tool.

In the current release, the Open Bootloader supports USART and USB protocols.

Here is the list of references to the user documents:

- [Flashlayout](https://wiki.st.com/stm32mpu/wiki/STM32CubeProgrammer_flashlayout) : Flashlayout management
- [OTP](https://wiki.st.com/stm32mpu/wiki/STM32CubeProgrammer_OTP_management) : OTP management
